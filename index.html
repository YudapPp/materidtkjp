1. Mind Mapping Komponen Komputer

Mind Mapping adalah teknik visual untuk menata informasi secara hierarkis, menunjukkan hubungan antara bagian-bagian dari suatu keseluruhan.

Klik pada setiap node untuk melihat penjelasan detail tentang komponen komputer tersebut.

Detail Komponen

Pilih salah satu komponen di mind map untuk melihat penjelasan detailnya.

Komputer terdiri dari tiga komponen utama: Hardware (perangkat keras), Software (perangkat lunak), dan Brainware (pengguna).

Hardware (Perangkat Keras)

Hardware adalah komponen fisik komputer yang dapat dilihat dan disentuh.

  • Input Device: Perangkat masukan (keyboard, mouse, scanner)
  • Processing Device: Perangkat pemrosesan (CPU, motherboard)
  • Output Device: Perangkat keluaran (monitor, printer, speaker)
  • Storage Device: Perangkat penyimpanan (harddisk, SSD, flashdisk)

Software (Perangkat Lunak)

Software adalah program yang berisi instruksi untuk menjalankan tugas tertentu pada komputer.

  • Sistem Operasi: Windows, Linux, macOS, Android
  • Aplikasi: Microsoft Office, Photoshop, Browser
  • Utility: Antivirus, Disk Cleanup, Backup Tools
  • Bahasa Pemrograman: Python, Java, C++, JavaScript

Brainware (Pengguna)

Brainware adalah orang yang menggunakan atau mengoperasikan komputer.

  • Operator: Menjalankan sistem komputer
  • Programmer: Membuat program/aplikasi
  • System Analyst: Menganalisis sistem yang dibutuhkan
  • Network Administrator: Mengelola jaringan komputer
